# Skill: Premium Website Builder (Inspiration → Clone → Refine → Deploy)

## Purpose
Build premium, designer-quality websites with Claude Code using a 4-step methodology:
visual inspiration first, structured clone second, component-level refinement third, deploy fourth.
Never prompt from nothing — always give Claude a visual + code reference to work from.

Resources:
- Inspiration galleries: https://godly.website | https://dribbble.com | https://awwwards.com
- Premium components: https://21st.dev | https://codepen.io/trending
- Animation techniques: see web-animations skill

## Step 0 — Gather Inputs
- inspiration_url, inspiration_screenshots (3-5), inspiration_css (DevTools → Styles), product_brief

## Step 1 — Inspiration Analysis
Extract: color palette (hex), typography, layout pattern, motion style, component inventory

## Step 2 — Build (Clone First)
Prompt: "Build React + Vite + TypeScript + Tailwind. REFERENCE: [url]. Pixel-accurate clone.
Match: colors, typography, layout, animation, background. CSS REFERENCE: [paste css]"

## Step 3 — Customize Brand
"Keep design system. Change only: copy, brand colors, assets.
PRODUCT / AUDIENCE / CTA / TONE / BRAND COLORS"

## Step 4 — Refine with 21st.dev
Priority order: Background → Hero → Feature cards → Buttons → Social proof strip
One component at a time. Build → preview → next.

## Step 5 — Deploy
GitHub: "Push to new repo [project-slug] using my GitHub account"
Vercel: vercel.com → import repo → Deploy

## Quality Standards
- Custom animated background
- Scroll-triggered animations
- Glass morphism cards
- Weighted CTA buttons with hover
- 8px spacing grid
- Mobile responsive
