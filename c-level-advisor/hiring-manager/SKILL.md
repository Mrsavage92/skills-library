# Hiring Manager

## Overview

Great hiring is a competitive advantage. This skill covers the full hiring lifecycle for technical roles: writing magnetic job descriptions, designing structured interview loops, building evaluation rubrics, calibrating offers, and improving hiring pipeline health.

## When to Use

- Writing or improving a technical job description
- Designing a structured interview loop for a new role
- Building evaluation rubrics and scoring guides
- Calibrating an offer against market data
- Improving offer acceptance rate or time-to-hire
- Diagnosing hiring funnel drop-off

## Interview Loop Design (Engineering)

### Standard 4-Stage Loop
| Stage | Format | Duration | Evaluates |
|-------|--------|----------|-----------|
| 1. Recruiter screen | Video call | 30 min | Motivation, fit, logistics |
| 2. Technical screen | Live coding | 60 min | Problem solving, communication |
| 3. System design | Whiteboard/Miro | 60 min | Architecture thinking |
| 4. Values/culture | Panel (2–3 people) | 45 min | Values alignment, collaboration |

### Evaluation Rubric (per stage)
Score 1–4 on each dimension:
- **1 — Strong No Hire:** Does not meet bar
- **2 — No Hire:** Below bar but shows potential
- **3 — Hire:** Meets bar
- **4 — Strong Hire:** Exceeds bar, would raise team level

### Structured Debrief Process
1. Each interviewer submits written feedback before debrief
2. Hiring manager reads all feedback, identifies divergence
3. 30-min debrief: start with most divergent scores
4. Reach consensus on hire/no-hire before discussion (avoid anchoring)
5. Hiring manager makes final decision with recruiter

## Offer Construction Framework
| Component | Considerations |
|-----------|---------------|
| Base salary | P50 market for level + location |
| Equity | % of company × current 409A valuation |
| Signing bonus | Used to bridge comp gap or RSA cliff |
| Benefits | Health, pension, remote allowance |
| Start date | Candidate's notice period + buffer |

## Hiring Funnel Benchmarks
| Stage | Typical Conversion |
|-------|-------------------|
| Application → Phone screen | 10–20% |
| Phone screen → Technical | 30–50% |
| Technical → Onsite/panel | 40–60% |
| Panel → Offer | 30–50% |
| Offer → Accept | 70–90% |

## Key Metrics
- **Time-to-hire** — days from role open to offer accepted (target: < 30 days)
- **Offer acceptance rate** — target > 80%
- **Funnel drop-off by stage** — identify where candidates are lost
- **Quality of hire** — 6-month performance rating of hires
- **Diversity metrics** — representation at each funnel stage
